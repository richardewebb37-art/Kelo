### Required Backgrounds
Please upload your construction blueprint background images to: `assets/images/backgrounds/`

### Recommended Specifications
- **Format**: JPG or PNG
- **Resolution**: 1920x1080 or higher (for desktop)
- **File Size**: Optimized for web (< 500KB per image)
- **Quantity**: 4-6 different variations

### Background Types Needed

1. **Primary Blueprint Background**
   - Classic deep blue blueprint paper
   - White grid lines visible
   - Technical drawing aesthetic
   - Use: Main background for most pages

2. **Light Technical Drawing**
   - Light cream/white paper
   - Blue grid lines
   - Use: Reading pages, forms, light sections

3. **Construction Site Texture**
   - Concrete or industrial texture
   - Optional yellow caution stripes
   - Use: Accent sections, headers

4. **Aged/Vintage Blueprint**
   - Worn, vintage appearance
   - Faded colors
   - Use: Special sections, about page

5. **Grid Overlay (Optional)**
   - Transparent PNG with just grid lines
   - Can be overlaid on solid colors
   - Use: Flexible background option

### Naming Convention
Please name your files descriptively:
- `blueprint-primary.jpg`
- `blueprint-light.jpg`
- `construction-texture.jpg`
- `blueprint-vintage.jpg`
- `grid-overlay.png`

## ðŸ’¡ Tips for Asset Selection

### For Backgrounds
- Choose high-contrast blueprints for readability
- Ensure grid lines are visible but not overwhelming
- Test on different screen sizes
- Consider file size for performance

### For Buttons
- Ensure buttons are clearly clickable
- Maintain consistent style across all buttons
- Include hover states for better UX
- Test on mobile devices

### For Icons
- Keep icons simple and recognizable
- Maintain consistent stroke width
- Ensure icons work at small sizes
- Test contrast against backgrounds

### For Fonts
- Ensure fonts are legible at small sizes
- Check licensing for web use
- Test on different devices
- Provide fallback fonts
